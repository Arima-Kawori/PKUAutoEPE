# PKUAutoEPE

By ArimaKawori, original code credit to https://github.com/Bruuuuuuce/PKUAutoSubmit.

## Requirements:

Python: 3.6+

Packages: selenium: 2.48.0+

```
pip3 install selenium
```

## Usage

1. Copy `config.sample.ini` to `config.ini`, and change the config in it.

2. Run main.py.
